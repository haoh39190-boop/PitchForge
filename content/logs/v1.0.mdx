PitchForge now supports a complete customer follow-up workflow:

- Create and edit customer profiles
- Capture vehicle interest, budget, stage, priorities, and main concerns
- Generate structured language from the current conversation context
- Edit the full script and rewrite it with a new phrasing
- Copy language and save communication records as separate actions
- Save polished versions to a personal script library

All content remains in the current browser so work can continue the next time it is opened.
